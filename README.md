# 📚 Estruturas Lineares

Repositório contendo projetos e exercícios desenvolvidos durante a disciplina de **Estruturas Lineares** do curso de Sistemas de Informação.

---

## 📌 Sobre

Este repositório reúne implementações práticas dos principais conceitos estudados na disciplina, utilizando Python e interface gráfica em alguns projetos.

O objetivo é consolidar conhecimentos sobre estruturas de dados fundamentais e algoritmos.

---

## 📖 Conteúdo

- Listas
- Pilhas
- Filas
- Listas Encadeadas
- Algoritmos de manipulação de dados
- Exercícios práticos
- Projetos desenvolvidos durante a disciplina

---

## 🛠️ Tecnologias

- Python
- Tkinter
- SQL (quando aplicável)

---

## 🎯 Objetivo

Aplicar na prática os conceitos de estruturas lineares, desenvolvendo soluções para problemas propostos durante a graduação.

---

```bash
python main.py
```
